Conceitos da Web Audio API

A Web Audio API utiliza um **Grafo de Áudio** que é composto por **Nós (Nodes)** conectados:

  • **`AudioContext`**: O contêiner de áudio principal, onde todos os nós vivem.
  • **`OscillatorNode`**: A fonte de som, que gera ondas sonoras (neste exemplo).
  • **`GainNode`**: Um nó de processamento que controla o volume (ganho).
  • **`context.destination`**: O destino final, geralmente os alto-falantes do seu dispositivo.
